### 常用命令

清理项目：
> gradle clean

构建项目、重新编译项目：
> gradle build

打war包：
> gradle war

打jar包：
> gradle jar

生成eclipse项目配置：
> gradle eclipse

清理eclipse相关配置：
> gradle cleanEclipse

查看项目的属性：
> gradle properties

查看项目依赖：
> gradle dependencies
